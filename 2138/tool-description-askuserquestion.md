Ask questions during execution to gather preferences, clarify instructions, or make implementation decisions.

Usage notes:
- Users can select "Other" for custom input
- multiSelect: true allows multiple answers
- Mark recommended option with "(Recommended)"
- In plan mode: clarify requirements BEFORE finalizing plan. Do NOT ask "Is my plan ready?"
