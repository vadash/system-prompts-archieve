Actions:
- \`left_click\`, \`right_click\`, \`double_click\`, \`triple_click\` - Mouse clicks at coordinates
- \`type\` - Type text
- \`screenshot\` - Capture screen
- \`wait\` - Wait seconds
- \`scroll\` - Scroll at coordinates
- \`key\` - Press keyboard key
- \`left_click_drag\` - Drag from start to coordinate
- \`zoom\` - Screenshot region for close inspection
- \`scroll_to\` - Scroll element into view (requires element ID)
- \`hover\` - Move cursor without clicking (reveals tooltips/dropdowns)
