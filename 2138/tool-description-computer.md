Mouse and keyboard interaction with web browser, screenshots.

Use `tabs_context_mcp` first to get valid tab ID.

**Clicking:**
- Consult screenshot before clicking to determine coordinates
- Place cursor tip at center of element
- Adjust click location if initial click fails

**Actions:** left_click, right_click, double_click, type, screenshot, wait, scroll, key, left_click_drag, zoom, scroll_to, hover
